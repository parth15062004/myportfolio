Gas Leakage Prevention

Project Overview

This Arduino-based LPG gas leakage prevention system detects leaks using an MQ-6 gas sensor and automatically shuts off the gas supply by controlling a servo motor on the regulator. It also triggers buzzer and LED alerts for safety.

Tech Used:

  • Arduino Uno
  • MQ-6 sensor
  • Servo Motor
  • Buzzer
